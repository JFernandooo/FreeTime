## O que é o FreeTime?

FreeTime é uma aplicação Android para gerir lembretes, tarefas e hábitos do dia-a-dia. Funciona **100% offline** — os teus dados ficam no teu telemóvel, encriptados. A sincronização com Google Drive é opcional.

---

## Funcionalidades

| Módulo | Descrição |
|--------|-----------|
| **Lembretes** | Prazos com notificação antecipada, modo nagging, alarme exacto |
| **Tarefas** | Lista de tarefas com subtarefas e tags |
| **Medicamentos** | Lembretes diários de doses com horários múltiplos |
| **Compras** | Listas de compras reutilizáveis |
| **Hábitos** | Acompanhamento de hábitos com lembrete diário |
| **Notas** | Notas rápidas com cores e fixação |

**Outras funcionalidades:**
- Calendário mensal e semanal
- Dashboard com resumo do dia
- Sincronização Google Drive (backup opcional)
- Modo multi-calendário
- Exportar para iCal
- Widgets para o ecrã inicial (3 tamanhos)
- Verificação automática de actualizações

## Instalar no Android

> A aplicação não está na Play Store. Instalação manual via APK.

1. **Descarrega o APK** na secção [Releases](https://github.com/JFernandooo/FreeTime/releases/latest)
2. No Android, vai a **Definições → Segurança → Fontes desconhecidas** e activa a opção (ou permite quando o sistema perguntar)
3. Abre o ficheiro APK descarregado
4. Segue o assistente de instalação
5. Abre o FreeTime e concede as permissões de notificações

---

## Privacidade

- Os dados ficam **apenas no teu dispositivo** (Hive, encriptado com AES-256)
- Sem conta obrigatória, sem servidores externos
- O backup no Google Drive é opcional e controlado por ti
- Nenhum dado é partilhado com terceiros

---

## Actualizações

A aplicação verifica actualizações automaticamente. Quando estiver disponível uma nova versão, aparece um aviso dentro da app com opção de instalar.
